Semantic HTML

Semantic html is using specific tags that give some description about the content within, Thus improving SEO ranking.

Tags Table

Tags Name Description
<nav> Navigation To link to different sections of the webpage.
<section> Section To markup different regions of the webpage.
<article> Article Specifies independent, self contained content.
<img> Image To display Images on the webpage.
<video> Video To display Videos on the webpage.

Attributes Table

Attribute Name Description
src Source To specify the location of the media to be imported.
alt Alternate Text Captions the image if the image fails to load.
